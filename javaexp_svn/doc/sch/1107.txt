### CSS & jQuery 기본 선택자
#### 1. 흥미로운 도입
CSS는 웹페이지의 스타일을 정의하는 언어입니다. 하지만 때로는 페이지의 
다양한 요소들에 동일한 스타일을 적용하고 싶을 때가 있습니다. 
이때 선택자가 활약하는 순간입니다. jQuery에서도 유사한 선택자를 사용하여 
DOM 요소를 쉽게 조작할 수 있죠. 상상해보세요, 한 번의 클릭으로 페이지의 
모든 이미지 테두리에 애니메이션을 적용한다면 얼마나 멋질까요?
#### 2. 학습목표
- CSS 선택자의 기본을 이해하고 사용할 수 있다.
- jQuery 선택자를 사용하여 DOM 요소를 선택하고 조작할 수 있다.
- CSS와 jQuery 선택자의 차이점과 공통점을 설명할 수 있다.
#### 3. 주요 핵심 키워드와 개념
- **CSS 선택자**: 태그, 클래스, ID, 속성 선택자 등
- **jQuery 선택자**: `$('selector')`를 사용한 요소, 클래스, ID 선택
- **유니버설 선택자**: 모든 요소를 선택하는 `*` 선택자
- **계층 선택자**: 자식 `>` 또는 후손 공백을 사용한 선택자
- **그룹 선택자**: 쉼표로 여러 선택자를 그룹화하는 방법
#### 4. 생각하고 토론해볼 내용(마지막에 조별로 정리해서 조장님이 전달해주세요)
- CSS 선택자와 jQuery 선택자 사이의 성능 차이는 어떤 의미가 있을까요?
- 왜 jQuery를 사용할 때 여전히 CSS 선택자를 알고 있어야 할까요?
- 클래스와 ID 선택자를 사용할 때의 장단점은 무엇일까요?
#### 5. 간단한 퀴즈(마지막에 조별로 정리해서 조장님이 전달해주세요)
1. `$(".class")`와 `$("#id")`는 jQuery에서 각각 어떤 요소를 선택하나요?
2. CSS에서 어떻게 하면 태그 p에 있는 모든 클래스가 `.class`인 자식 요소들을 
   선택할 수 있나요?
3. jQuery에서 `$("ul li:first-child")`는 무엇을 의미하나요?
#### 6. 실무적용 예시
- **웹사이트 내비게이션 바 스타일링**: CSS 클래스 선택자를 사용하여 사이트의 모든 
    내비게이션 바 항목에 공통 스타일을 적용합니다.
- **폼 유효성 검사 애니메이션**: jQuery를 사용하여 폼 입력이 유효하지 않을 때 
    해당 입력 필드에 애니메이션 효과를 추가합니다.
- **갤러리 이미지 강조**: 갤러리 내의 첫 번째 이미지에 특별한 스타일을 적용하여 
    사용자의 주의를 끕니다.

### 오라클 그룹함수 
#### 1. 흥미로운 도입
- **데이터의 파워**: 대량의 데이터를 분석하여 인사이트를 얻는 사례 소개. 예를 들면, 
많은 사람들이 웹사이트를 방문하면 어떤 상품을 가장 많이 구매하는지, 어떤 지역에서 가장 
많은 판매가 이루어지는지 등의 정보를 얻기 위해 그룹 함수를 사용하는 경우.
#### 2. 학습목표
- 그룹 함수의 정의와 사용법 이해
- 주요 그룹 함수의 종류와 특징 학습
- 실무 데이터 분석에서 그룹 함수의 적용 방법 습득
#### 3. 주요 핵심 키워드와 개념
- **그룹 함수(Group Function)**: 여러 행의 데이터를 그룹화하여 결과를 도출하는 함수
- **주요 함수**: `SUM`, `AVG`, `MAX`, `MIN`, `COUNT`, `GROUP BY`, 
               `HAVING`
#### 4. 생각하고 토론해볼 내용
- 그룹 함수의 필요성: 단순한 데이터 조회 이상의 분석이 필요한 이유는?
- 그룹 함수의 한계와 확장 가능성: 다른 도구나 방법론과 결합하여 분석하는 방법은?
#### 5. 간단한 퀴즈
1. `SUM` 함수의 용도는?
2. `GROUP BY`와 `HAVING`의 차이점은?
3. 특정 부서의 평균 급여를 조회하려면 어떤 그룹 함수를 사용해야 하는가?
#### 6. 실무적용 예시
- **매출 분석**: 월별/지역별 매출 집계 및 분석
- **고객 분석**: 고객별/제품별 구매 통계 및 평균 구매 금액 분석
- **재고 관리**: 상품 카테고리별 재고량 및 평균 입고 시간 분석
