css 시작
### 1. 흥미로운 도입
"웹 페이지는 그 자체로 한 권의 책과도 같습니다. 하지만 CSS가 없다면, 그것은 단지 
글자가 뒤섞인 백지장에 불과해요. 이제 우리는 그 백지장에 색을 입히고, 레이아웃을 구성하여
 사용자에게 아름답고 명확한 메시지를 전달할 수 있는 예술 작품으로 변모시킬 것입니다."
### 2. 학습목표
- CSS의 기본 구조와 사용 방법을 이해합니다.
- 주요 CSS 속성에 대한 지식을 갖추고, 이를 통해 웹 페이지를 스타일링할 수 있습니다.
- 반응형 디자인을 구현하기 위한 미디어 쿼리의 기본을 배웁니다.
- 실무에서 자주 사용되는 CSS 프레임워크의 기본적인 사용법을 알아봅니다.
### 3. 주요 핵심 키워드와 개념
- **선택자(Selectors)**: 요소, 클래스, ID 선택자
- **속성(Properties)**: `color`, `font-family`, `margin`, `padding`
- **박스 모델(Box Model)**: 마진, 보더, 패딩, 콘텐츠 영역
- **레이아웃(Layout)**: `display`, `position`, `flexbox`, `grid`
- **단위(Units)**: 픽셀(px), em, rem, vh, vw
- **응답성(Responsiveness)**: 미디어 쿼리(media queries), 모바일 우선 접근법
- **애니메이션(Animations)** 및 **트랜지션(Transitions)**
### 4. 생각하고 토론해볼 내용
- 왜 CSS는 웹 개발에서 중요한가요?
- CSS를 통해 사용자 경험에 어떤 영향을 줄 수 있나요?
- 모바일 우선 접근법(mobile-first approach)이란 무엇이고 왜 중요할까요?
- CSS 프레임워크는 어떻게 웹 개발 프로세스를 간소화할 수 있나요?
### 5. 간단한 퀴즈
1. CSS에서 "Cascading"이란 무엇을 의미하나요?
2. 박스 모델의 네 가지 주요 구성 요소는 무엇인가요?
3. `flex`와 `grid`의 차이점은 무엇인가요?
4. `em`과 `rem` 단위의 차이점은 무엇인가요?
5. 미디어 쿼리란 무엇이며, 어떻게 사용되나요?

### 6. 실무적용 예시

- **레이아웃 디자인**: 플렉스박스를 사용하여 내비게이션 바를 만들기
- **응답형 웹 페이지**: 미디어 쿼리를 사용하여 다양한 화면 크기에 대응하는 레이아웃 구성하기
- **테마 적용**: CSS 변수를 사용하여 웹 사이트 테마를 빠르게 변경할 수 있는 시스템 구축
- **애니메이션 효과**: 버튼에 호버 효과를 주어 사용자의 인터랙션을 유도하기
